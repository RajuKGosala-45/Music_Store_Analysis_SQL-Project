# Music Store SQL Analysis

## Overview
This project analyzes the Music Store database using SQL to understand customer spending, revenue trends, genre popularity, and artist performance. It includes Easy, Moderate, and Advanced SQL queries.

## Key Highlights
- Identified top customers and highest revenue cities.
- Analyzed most popular genres and top-performing artists.
- Used JOINs, subqueries, aggregates, CTEs, and window functions.
- Worked with Customer, Invoice, InvoiceLine, Track, Album, Artist, and Genre tables.
